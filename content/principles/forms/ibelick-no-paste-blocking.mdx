<Callout type="quote" title="UI Skills Rule">
  NEVER block paste in form fields. It breaks password managers, accessibility tools, and frustrates users.
</Callout>

## Why This Matters

Blocking paste:
- Breaks password managers (the #1 security tool)
- Prevents users with motor disabilities from using assistive tools
- Forces users to type sensitive data (increasing errors)
- Annoys power users who copy/paste long values

The "confirm email" anti-pattern makes this worse by blocking paste on re-entry fields.
