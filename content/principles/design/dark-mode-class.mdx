<Callout type="quote" title="Tailwind Golden Rule">
  Use class-based dark mode (darkMode: 'class') for user-controllable themes. This allows toggling between light/dark/system preferences.
</Callout>

## Why This Matters

The `media` strategy only responds to OS settings - users can't override it. The `class` strategy lets you add/remove the `dark` class on `<html>`, enabling theme toggles and respecting user choice over system preference.

## Implementation

<CodeBlock title="Configuration" language="javascript">
{`// tailwind.config.js
module.exports = {
  darkMode: 'class', // or 'selector' in v4
}

// HTML structure
<html class="dark">
  <!-- dark: variants now work -->
</html>

// Toggle with JavaScript
document.documentElement.classList.toggle('dark');`}
</CodeBlock>
