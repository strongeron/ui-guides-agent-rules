<Callout type="quote" title="From the Guidelines">
  Layered shadows. Mimic ambient + direct light with at least two layers.
</Callout>

## Why This Matters

Natural shadows have both ambient (soft, diffuse) and direct (sharper, darker) components. A single box-shadow looks flat and artificial.

Use two box-shadow declarations: one large and subtle for ambient light, one smaller and darker for direct light. This creates depth and realism that makes elements feel like they exist in physical space.

<CodeBlock title="CSS Implementation" language="css">
{`.card {
  /* Two-layer shadow for natural depth */
  box-shadow:
    /* Ambient: large, soft, light */
    0 8px 24px rgba(0, 0, 0, 0.06),
    /* Direct: smaller, sharper, darker */
    0 2px 8px rgba(0, 0, 0, 0.08);
}

.elevated-card {
  /* More elevation = more shadow separation */
  box-shadow:
    0 16px 48px rgba(0, 0, 0, 0.08),
    0 4px 12px rgba(0, 0, 0, 0.1);
}`}
</CodeBlock>

<CodeBlock title="Tailwind v4 Custom Theme" language="css">
{`/* In your CSS file with @theme */
@theme {
  --shadow-layered: 0 1px 3px rgba(0, 0, 0, 0.05), 0 10px 25px rgba(0, 0, 0, 0.1);
  --shadow-layered-lg: 0 2px 8px rgba(0, 0, 0, 0.06), 0 16px 48px rgba(0, 0, 0, 0.08);
}

/* Usage */
<div class="shadow-layered">Card with layered shadow</div>
<div class="shadow-layered-lg">Elevated card</div>`}
</CodeBlock>
