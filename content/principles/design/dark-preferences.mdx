<Callout type="quote" title="Tailwind Golden Rule">
  Read prefers-color-scheme on initial load and apply the dark class before first paint to prevent flash of wrong theme.
</Callout>

## Why This Matters

If you wait for JavaScript to run before checking the color scheme, users see a flash of the wrong theme (FART - Flash of inAccurate coloR Theme). Inline a script in `<head>` that runs synchronously before render.

## Implementation

<CodeBlock title="Inline Script" language="html">
{`<head>
  <script>
    // Runs before first paint
    const theme = localStorage.getItem('theme');
    const prefersDark = matchMedia('(prefers-color-scheme: dark)').matches;

    if (theme === 'dark' || (!theme && prefersDark)) {
      document.documentElement.classList.add('dark');
    }
  </script>
</head>`}
</CodeBlock>
