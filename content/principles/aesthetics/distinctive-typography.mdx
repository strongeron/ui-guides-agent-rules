## Why This Matters

Typography carries singular voice. Generic fonts like Inter, Roboto, and system
fonts are everywhere—they're safe but forgettable. Distinctive font pairings
create immediate visual identity and memorability.

The right typography pairing—a characterful display font for headlines with a
refined body font—establishes brand personality before users read a single word.

## Best Practices

- **Display fonts** for headlines: Playfair Display, Fraunces, Space Grotesk
- **Body fonts** with character: Source Serif, Literata, IBM Plex
- Use `font-display: swap` to prevent invisible text during load
- Match fallback font metrics with `size-adjust` to minimize CLS
