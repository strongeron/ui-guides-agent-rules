## Why This Matters

Static content is forgettable. When all elements appear immediately, there is
no journey, no discovery, no delight. Scroll-triggered animations transform
passive browsing into active exploration.

Each reveal becomes a moment of anticipation. Users feel rewarded for scrolling.
The interface responds to their input, creating a dialogue between user and design.

## Best Practices

- Use `IntersectionObserver` for performant scroll detection
- Stagger animations with `animation-delay` (100-150ms intervals)
- Keep animations subtle - translate 10-20px, fade from 0 to 1
- Use `motion-safe:` to respect user preferences for reduced motion
- Reveal elements in natural reading order (top to bottom, left to right)
- Avoid revealing too many elements at once - let users discover progressively

## Code Pattern

```tsx
const observer = new IntersectionObserver(
  (entries) => {
    entries.forEach((entry) => {
      if (entry.isIntersecting) {
        entry.target.classList.add('is-visible');
      }
    });
  },
  { threshold: 0.3 }
);
```
