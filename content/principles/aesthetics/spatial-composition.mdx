## Why This Matters

Symmetric grids are the default. Every SaaS landing page, every dashboard,
every e-commerce site—they all reach for the same 3-column grid. Breaking
these expectations creates memorable, distinctive experiences.

Asymmetry, overlap, and varied scale aren't chaos—they're composition.
Use them with intention to guide the eye and create hierarchy.

## Best Practices

- Feature one dominant element at unexpected scale
- Use overlap to create depth (negative margins, absolute positioning)
- Break alignment deliberately—but align everything else precisely
- Consider diagonal flow for dynamic energy
- Use whitespace asymmetrically to create tension
