## Why This Matters

Craftsmanship lives in details users feel but rarely consciously notice:
consistent spacing rhythms, pixel-perfect alignment, harmonious proportions.
These details accumulate into an impression of quality and trust.

Sloppy execution—misaligned elements, inconsistent padding, arbitrary values
like 13px or 17px—destroys perceived quality. Users may not articulate why
something feels "cheap" or "broken," but they sense it immediately.

## Craftsmanship Checklist

- **Spacing scale**: Use a consistent system (4/8/12/16/24/32/48px)
- **Type scale**: Stick to established sizes (sm/base/lg/xl/2xl)
- **Border radii**: Use consistent values (md/lg/xl) that relate to each other
- **Alignment**: Every element should snap to a grid or intentional offset
- **Proportions**: Related elements should have mathematical relationships

## Common Anti-Patterns

- Arbitrary pixel values (11px, 17px, 31px) outside any scale
- Different padding on each side without visual justification
- Icons or elements that don't align with adjacent text
- Border radii that vary without reason (6px, 8px, 12px on siblings)
- Spacing that changes between similar components
