## Why This Matters

Motion is choreography. When animations fire randomly, the interface feels
broken or amateurish. Orchestrated reveals—elements appearing in sequence
with consistent timing—communicate intentionality and craftsmanship.

Think of it like a well-edited film: each cut, each reveal serves the story.
Your interface should feel the same way.

## Best Practices

- Use `animation-delay` with consistent intervals (100-150ms between items)
- Reveal in reading order: hero first, then navigation, then content
- Keep individual animations short (200-400ms)
- Use `motion-safe:` to respect user preferences
- Consider `@starting-style` for entry animations in CSS
