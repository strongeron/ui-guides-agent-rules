<Callout type="quote" title="Tailwind Golden Rule">
  Install Tailwind CSS IntelliSense in your IDE. It provides autocomplete, syntax highlighting, hover previews, and linting for class names.
</Callout>

## Why This Matters

Without IntelliSense, you're typing class names from memory, making typos, and missing utilities. IntelliSense shows available classes, their CSS output, color swatches, and warns about errors - dramatically improving productivity.

## Implementation

<CodeBlock title="VS Code Setup" language="json">
{`// Install: Tailwind CSS IntelliSense (bradlc.vscode-tailwindcss)

// settings.json - Enable in more places
{
  "tailwindCSS.includeLanguages": {
    "typescript": "javascript",
    "typescriptreact": "javascript"
  },
  "editor.quickSuggestions": {
    "strings": true
  },
  "tailwindCSS.classAttributes": [
    "class",
    "className",
    "tw"
  ]
}`}
</CodeBlock>
