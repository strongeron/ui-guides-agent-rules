<Callout type="quote" title="WCAG 2.2.2 Pause, Stop, Hide">
  Auto-playing animations >5s MUST have pause/stop/hide controls; OR auto-stop after 5s. Applies to carousels, video backgrounds, infinite loops. Essential animations (loading spinners, progress) are exempt.
</Callout>

## Why This Matters

WCAG 2.2.2 Level A requires that users can pause, stop, or hide moving content. This prevents distraction for users with attention disorders (ADHD), cognitive disabilities, and reduces seizure risk for users with photosensitive epilepsy.

Loading indicators and progress bars are exempt as they provide essential feedback about system status.

## Implementation

  badKey="animations-pause-stop-hide-bad"
  goodKey="animations-pause-stop-hide-good"

<CodeBlock title="Good Implementation">
```tsx
function AutoPlayCarousel() {
  const [isPaused, setIsPaused] = useState(false);

  return (
    <div>
      <div className="carousel" aria-live={isPaused ? 'polite' : 'off'}>
        {/* Carousel content */}
      </div>
      <button
        onClick={() => setIsPaused(!isPaused)}
        aria-label={isPaused ? 'Play carousel' : 'Pause carousel'}
      >
        {isPaused ? 'Play' : 'Pause'}
      </button>
    </div>
  );
}
```
</CodeBlock>
