<Callout type="quote" title="Modern Page Transitions">
  Use View Transitions API for page/state transitions. Add `@view-transition { navigation: auto; }` for MPA. Use `view-transition-name` for morphing elements. Progressive enhancement—works without JS.
</Callout>

## Why This Matters

The View Transitions API enables smooth animated transitions between DOM states or pages. It works for both SPAs and MPAs, can morph elements across states, and degrades gracefully in unsupported browsers.

This eliminates the need for complex animation libraries just for page transitions.

<CodeBlock title="MPA View Transitions (CSS)">
```css
@view-transition {
  navigation: auto;
}

/* Customize the transition */
::view-transition-old(root),
::view-transition-new(root) {
  animation-duration: 300ms;
}

/* Morph specific elements */
.hero-image {
  view-transition-name: hero;
}
```
</CodeBlock>

<CodeBlock title="SPA View Transitions (JS)">
```tsx
async function navigateWithTransition(url: string) {
  if (!document.startViewTransition) {
    // Fallback for unsupported browsers
    window.location.href = url;
    return;
  }

  const transition = document.startViewTransition(async () => {
    // Update DOM here
    await loadNewContent(url);
  });

  await transition.finished;
}
```
</CodeBlock>
