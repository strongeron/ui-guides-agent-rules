<Callout type="quote" title="UI Skills Rule">
  Pause looping animations when not visible. Use IntersectionObserver to detect visibility and stop wasting resources.
</Callout>

## Why This Matters

Animations running off-screen waste CPU cycles and drain battery on mobile devices. This is especially important for:
- Infinite loading spinners
- Background video/canvas animations
- CSS keyframe loops
- requestAnimationFrame loops
