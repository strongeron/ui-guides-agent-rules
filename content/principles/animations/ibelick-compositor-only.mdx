<Callout type="quote" title="UI Skills Rule">
  Animate only transform and opacity. These are compositor-thread properties that can achieve 60fps without triggering layout or paint.
</Callout>

## Why This Matters

The browser rendering pipeline has three main phases: Layout → Paint → Composite. Properties like `transform` and `opacity` only affect the composite phase, allowing the GPU to handle them on a separate thread. This means smooth animations even when the main thread is busy.
