<Callout type="quote" title="UI Skills Rule">
  NEVER animate width, height, top, left, margin, or padding. These trigger expensive layout recalculations on every frame.
</Callout>

## Why This Matters

Animating layout properties forces the browser to recalculate the position and size of potentially hundreds of elements on every frame. This causes jank, especially on lower-powered devices. Instead, use `transform: scale()` for size changes and `transform: translate()` for position changes.
