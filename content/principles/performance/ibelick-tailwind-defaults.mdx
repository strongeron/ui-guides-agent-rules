<Callout type="quote" title="UI Skills Rule">
  MUST use Tailwind CSS defaults (spacing, colors, sizing) unless a custom design token exists in the config.
</Callout>

## Why This Matters

Tailwind's default scale is carefully designed for consistency. Using arbitrary values like `p-[13px]` or `text-[17px]` creates visual inconsistency and makes your CSS harder to maintain. The default spacing scale (4, 8, 12, 16, 20, 24, etc.) aligns to a 4px grid that works well across components.
