<Callout type="quote" title="Tailwind Golden Rule">
  Animate only transform and opacity properties. These run on the GPU compositor thread and won't cause layout recalculations.
</Callout>

## Why This Matters

Animating properties like `width`, `height`, `top`, `left`, or `margin` triggers expensive layout recalculations on every frame. GPU-accelerated properties (`transform`, `opacity`) run on a separate compositor thread, achieving smooth 60fps animations even on slower devices.

## Implementation

<CodeBlock title="Tailwind Classes" language="html">
{`<!-- Bad: Animates margin (triggers layout) -->
<div class="hover:mt-2 transition-all">

<!-- Good: Uses transform (GPU-accelerated) -->
<div class="hover:-translate-y-1 transition-transform">`}
</CodeBlock>
