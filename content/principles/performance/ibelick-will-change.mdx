<Callout type="quote" title="UI Skills Rule">
  Only apply will-change during active animations. Permanent will-change wastes GPU memory and can hurt performance.
</Callout>

## Why This Matters

`will-change` tells the browser to create a GPU layer for an element. This consumes memory and has overhead. Using it permanently on many elements can actually hurt performance by:
- Consuming GPU memory
- Causing layer explosion
- Preventing the browser's own optimizations
