<Callout type="quote" title="Tailwind Golden Rule">
  Use @layer directives (base, components, utilities) to add custom CSS that respects Tailwind's specificity system and purging.
</Callout>

## Why This Matters

CSS added outside @layer directives has higher specificity than Tailwind utilities, making them impossible to override. @layer ensures your custom styles integrate properly with Tailwind's cascade and are included in purging.

## Implementation

<CodeBlock title="CSS" language="css">
{`/* Bad: Outside layers - can't be overridden by utilities */
.btn {
  padding: 0.5rem 1rem;
}

/* Good: In component layer */
@layer components {
  .btn {
    @apply px-4 py-2 rounded-lg font-medium;
  }
}

/* Good: Custom utility */
@layer utilities {
  .text-balance {
    text-wrap: balance;
  }
}`}
</CodeBlock>
