<Callout type="quote" title="UI Skills Rule">
  MUST use fixed z-index scale, no arbitrary z-*. Define semantic z-index layers (base, dropdown, sticky, modal, popover, tooltip).
</Callout>

## Why This Matters

Arbitrary z-index values like `z-[999]` or `z-[9999]` create an unmanageable stacking context. When you need something on top of `z-[9999]`, you end up with `z-[99999]`. A semantic scale ensures predictable stacking: tooltips above modals, modals above dropdowns, etc.
