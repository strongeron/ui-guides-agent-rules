<Callout type="quote" title="UI Skills Rule">
  Use size-* instead of separate w-* h-* for square elements. It's more concise and clearly communicates intent.
</Callout>

## Why This Matters

When an element should be square (icons, avatars, buttons), using `size-8` instead of `w-8 h-8` is more concise and communicates intent clearly. It's immediately obvious the element is meant to be square, and you only need to change one value to resize it.
