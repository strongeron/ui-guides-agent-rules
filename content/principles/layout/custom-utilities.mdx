<Callout type="quote" title="Tailwind Golden Rule">
  Create custom utilities in @layer utilities (or @utility in v4) for reusable single-purpose classes that Tailwind doesn't provide.
</Callout>

## Why This Matters

Some CSS properties aren't in Tailwind by default (text-wrap: balance, scrollbar styling, etc.). Adding them as utilities in the proper layer means they work with variants (hover:, dark:, md:) and get purged correctly.

## Implementation

<CodeBlock title="CSS" language="css">
{`/* Tailwind v3 */
@layer utilities {
  .text-balance {
    text-wrap: balance;
  }
  .scrollbar-hide {
    scrollbar-width: none;
    &::-webkit-scrollbar { display: none; }
  }
}

/* Tailwind v4 */
@utility text-balance {
  text-wrap: balance;
}

/* Now works with variants! */
<h1 class="md:text-balance">...</h1>`}
</CodeBlock>
