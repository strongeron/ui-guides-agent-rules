<Callout type="quote" title="UI Skills Rule">
  NEVER use h-screen (100vh) for full-height layouts. Use h-dvh (dynamic viewport height) to account for mobile browser chrome.
</Callout>

## Why This Matters

On mobile browsers, `100vh` includes the area behind the browser's address bar and navigation. This causes content to be cut off when the browser chrome is visible. `dvh` (dynamic viewport height) adjusts as the browser chrome shows/hides.
