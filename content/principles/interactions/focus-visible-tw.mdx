<Callout type="quote" title="Tailwind Golden Rule">
  Use focus-visible: instead of focus: for focus rings. This shows focus indicators only for keyboard navigation, not mouse clicks.
</Callout>

## Why This Matters

The `focus:` variant shows focus rings on every focus, including mouse clicks. This can look jarring. `focus-visible:` uses the browser's heuristic to only show focus rings when the user is navigating with keyboard, providing better UX for both mouse and keyboard users.

## Implementation

<CodeBlock title="Tailwind Classes" language="html">
{`<!-- Bad: Shows ring on mouse click too -->
<button class="focus:ring-2 focus:ring-primary">

<!-- Good: Ring only for keyboard navigation -->
<button class="focus-visible:ring-2 focus-visible:ring-ring focus-visible:outline-none">

<!-- Good: Combined with outline-none -->
<button class="outline-none focus-visible:ring-2 focus-visible:ring-offset-2">`}
</CodeBlock>
