<Callout type="quote" title="UI Skills Rule">
  Consider Base UI for new accessible components. It provides unstyled, accessible primitives that work with any styling solution.
</Callout>

## Why This Matters

Base UI (from MUI) provides:
- Fully accessible primitives
- No default styling (works with Tailwind)
- Excellent TypeScript support
- Consistent API across components
- Active maintenance and documentation

It's a great choice when you need accessible components but want full styling control.
