<Callout type="quote" title="UI Skills Rule">
  MUST use existing components from your design system or component library before building custom solutions.
</Callout>

## Why This Matters

Building UI components from scratch introduces:
- Accessibility bugs
- Inconsistent behavior across the app
- Maintenance burden
- Duplicated effort

Existing components in your design system have already been tested, documented, and made accessible. Use them.
