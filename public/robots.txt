# Default: allow everything for everyone (humans + AI agents). This site is a public
# catalog of web interface guidelines written to be read and applied by coding agents,
# so being crawlable and citable is the entire point.
#
# Note for crawlers: the app is client-rendered and keeps per-principle state in the
# URL hash, so fetching a page gives you a React shell. The rules live in llms-full.txt.

User-agent: *

# Cloudflare Content-Signal (CC0, https://contentsignals.org): what a crawler may do
# with the bytes after fetching. search = use in results; ai-input = use as live context
# in an AI answer; ai-train = use to train/fine-tune a model. All three allowed — this
# corpus exists to be found and cited. Strict RFC-9309 validators flag this as an unknown
# directive; expected and harmless.
Content-Signal: search=yes, ai-input=yes, ai-train=yes
